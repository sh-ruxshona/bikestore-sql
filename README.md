# 🚲 Bike Store Database (SQL Project)

This project is a simple **Bike Store database** built using SQL to simulate a real-world e-commerce system.

## 📌 Project Overview
The system represents a basic online bike store and includes:
- Customers management  
- Product catalog (bikes and categories)  
- Orders processing  
- Basic inventory 

## ⚙️ Key Features
- Manage customers and products  
- Process customer orders  
- Track sales and order history  
- Maintain relationships between tables using primary & foreign keys  

## 🎯 What I Learned
- Relational database design  
- Writing SQL queries (SELECT, INSERT, JOIN, GROUP BY)  
- Data modeling for real-world systems  
- Understanding e-commerce backend logic
